## Basic Cybersecurity Topics Covered

The basic-level topics included across the HTML syllabus files are:

- Introduction to cybersecurity
- Cyber threat landscape and common attacks
- CIA triad and core security principles
- Networking basics (OSI, TCP/IP, ports, DNS, HTTP/HTTPS)
- Linux fundamentals for security learners
- Windows security basics and hardening
- Password security and multi-factor authentication
- Ethical hacking fundamentals and legal boundaries
- Reconnaissance and OSINT basics
- Nmap scanning and enumeration
- Wireshark packet analysis
- Vulnerability scanning introduction
- Phishing awareness and social engineering basics
- Endpoint protection basics
- Web security fundamentals
- Burp Suite basics
- OWASP Top 10 introduction
- SQL Injection basics
- XSS basics
- IDOR and access control basics
- Secure coding and input validation basics
- Security policies and compliance introduction
- Incident response basics
- Log monitoring basics
- Risk assessment fundamentals
- Cybersecurity reporting and documentation
- Final practical and theory assessments

## 3-Month Advanced Track Highlights

The advanced syllabus in `3-Month VDP and Bug Bounty Advanced.html` includes:

- VDP and Bug Bounty methodology
